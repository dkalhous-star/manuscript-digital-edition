# Manuscript Digital Edition

## Project Description
This project aims to provide a digital edition platform for manuscripts, allowing users to explore, annotate, and interact with historical documents in a user-friendly interface.

## Setup Instructions
1. **Clone the Repository**  
   Use the following command to clone the repository:
   ```bash
   git clone https://github.com/dkalhous-star/manuscript-digital-edition.git
   ```  

2. **Install Dependencies**  
   Navigate to the project directory and install the required dependencies:
   ```bash
   cd manuscript-digital-edition
   npm install
   ```  

3. **Run the Project**  
   Start the local development server:
   ```bash
   npm start
   ```

## Adding Manuscripts
To add new manuscripts to the project:
1. Create a new folder inside the `manuscripts` directory for the manuscript you want to add.
2. Include all relevant files (images, text files, etc.) related to the manuscript in this folder.
3. Update the manuscript list in the project to include the new addition.
